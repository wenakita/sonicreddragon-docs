# Creating Elegant Interactive Diagrams

This guide demonstrates how to create beautiful, animated diagrams in your documentation using Mermaid and Anime.js.

## Basic Mermaid Diagram

Let's start with a basic flowchart that automatically animates:

<StandardMermaid
  chart={`
flowchart LR
    A[Start] --> B{Is it working?}
    B -->|Yes| C[Great!]
    B -->|No| D[Debug]
    D --> B
    C --> E[Deploy]
  `}
/>

## Using the EnhancedMermaid Component

To use the enhanced component in your MDX files:

```jsx
import EnhancedMermaid from '@site/src/components/EnhancedMermaid';

<EnhancedMermaid
  title="My Diagram"
  caption="Optional caption text"
  chart={`
    flowchart LR
      A --> B
      B --> C
  `}
/>
```

## Customizing Animations

For custom animations, you can pass animation options:

```jsx
<EnhancedMermaid
  animated={true}
  pulseSelector=".important-node"
  animationOptions={{
    delay: 1500,
    duration: 1000
  }}
  chart={`...`}
/>
```

## Using the StandardMermaid Component

If you're having issues with the standard Mermaid code blocks (using triple backticks), you can use the StandardMermaid component as a reliable alternative:

```jsx
import StandardMermaid from '@site/src/components/StandardMermaid';

<StandardMermaid
  chart={`
    flowchart LR
      A --> B
      B --> C
  `}
/>
```

With these components and techniques, you can create beautiful, interactive diagrams that enhance your documentation with elegant animations and visual appeal.
