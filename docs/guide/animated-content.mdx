# Creating Animated Content

This guide will show you how to add beautiful animations to your documentation using our anime.js integration.

## Introduction

Animations can make your documentation more engaging and help illustrate complex concepts. Our anime.js integration provides:

- **Smooth animations** - Elegant, performant animations that enhance the user experience
- **Scroll-triggered effects** - Elements that animate as they enter the viewport
- **Interactive elements** - Hover effects and click animations
- **Particle effects** - Beautiful background particles for immersive sections
- **Accessibility** - All animations respect the user's motion preferences

## Basic Animation Components

### Animated Text

You can animate text elements by adding the `data-animated-text` attribute:

```jsx
<h2 data-animated-text>This text will animate in</h2>
<p data-animated-text data-animation-delay="200">This text will animate in with a delay</p>
```

### Scroll Reveal

Elements with the `data-scroll-reveal` attribute will animate as they enter the viewport:

```jsx
<div data-scroll-reveal>
  <p>This content will fade in when scrolled into view</p>
</div>

<div data-scroll-reveal data-animation="slide-up">
  <p>This content will slide up when scrolled into view</p>
</div>
```

Available animation types:
- `fade-in` (default)
- `slide-up`
- `slide-down`
- `slide-left`
- `slide-right`
- `zoom-in`
- `zoom-out`

### Interactive Elements

Add the `interactive-element` class to create elements that respond to hover:

```jsx
<div className="interactive-element">
  <h3>Hover over me</h3>
  <p>I'll respond with a subtle animation</p>
</div>
```

### Animated Buttons

Add the `data-animated-button` attribute to create buttons with hover effects:

```jsx
<button data-animated-button>
  Click me
</button>

<a href="/some-page" data-animated-button>
  Learn more
</a>
```

## Animation Utilities

### Scroll Reveal Wrapper

Wrap any content with `ScrollRevealWrapper` to animate it when scrolled into view:

```jsx
import ScrollRevealWrapper from '@site/src/components/ScrollRevealWrapper';

<ScrollRevealWrapper animation="slide-up" delay={200}>
```

### Animation Provider

Use the `AnimationProvider` to control animations for a section of content:

```jsx
import AnimationProvider from '@site/src/components/AnimationProvider';

<AnimationProvider enabled={true} reducedMotion={false}>
  <div>
    <h2 data-animated-text>This text will animate</h2>
    <p data-animated-text>Even if the user has reduced motion enabled</p>
  </div>
</AnimationProvider>
```

## Best Practices

Here are some best practices for using animations effectively:

1. **Use animations purposefully** - Animations should enhance the content, not distract from it
2. **Keep it subtle** - Subtle animations are more effective than flashy ones
3. **Consider performance** - Too many animations can impact performance
4. **Respect user preferences** - Always respect the user's motion preferences
5. **Test on different devices** - Make sure animations work well on all devices
6. **Provide fallbacks** - Content should still be usable without animations

## Accessibility

All animations respect the user's motion preferences. If a user has enabled the "reduced motion" setting in their operating system, animations will be disabled or simplified.

You can test this by enabling the "reduced motion" setting in your operating system:
- **macOS**: System Preferences > Accessibility > Display > Reduce motion
- **Windows**: Settings > Ease of Access > Display > Show animations in Windows
- **iOS**: Settings > Accessibility > Motion > Reduce Motion
- **Android**: Settings > Accessibility > Remove animations

## Advanced Techniques

### Custom Animations with anime.js

You can create custom animations using the anime.js library directly:

```jsx
import React, { useEffect, useRef } from 'react';
import anime from 'animejs';

function CustomAnimation() {
  const elementRef = useRef(null);
  
  useEffect(() => {
    if (elementRef.current) {
      anime({
        targets: elementRef.current,
        translateY: [50, 0],
        opacity: [0, 1],
        duration: 1000,
        easing: 'easeOutCubic',
      });
    }
  }, []);
  
  return (
    <div ref={elementRef} style={{ opacity: 0 }}>
      <p>This element has a custom animation</p>
    </div>
  );
}
```

### Sequenced Animations

You can create sequenced animations by chaining anime.js animations:

```jsx
import React, { useEffect } from 'react';
import anime from 'animejs';

function SequencedAnimation() {
  useEffect(() => {
    const timeline = anime.timeline({
      easing: 'easeOutExpo',
    });
    
    timeline
      .add({
        targets: '.step-1',
        translateY: [50, 0],
        opacity: [0, 1],
        duration: 500,
      })
      .add({
        targets: '.step-2',
        translateY: [50, 0],
        opacity: [0, 1],
        duration: 500,
      })
      .add({
        targets: '.step-3',
        translateY: [50, 0],
        opacity: [0, 1],
        duration: 500,
      });
  }, []);
  
  return (
    <div>
      <div className="step-1" style={{ opacity: 0 }}>Step 1</div>
      <div className="step-2" style={{ opacity: 0 }}>Step 2</div>
      <div className="step-3" style={{ opacity: 0 }}>Step 3</div>
    </div>
  );
}
```
