# Creating Animated Content

Our documentation system includes built-in support for creating animated content using [anime.js](https://animejs.com/), a lightweight JavaScript animation library. This guide will show you how to use our pre-built components and create custom animations to make your documentation more engaging and interactive.

  size="xl"

## Using Pre-built Animation Components

We provide several pre-built components that you can use to add animations to your documentation:

### Animated Text

The `AnimatedText` component allows you to create text with various animation effects:

```jsx
import AnimatedText from '@site/src/components/AnimatedText';

<AnimatedText
  text="This text animates on page load"
  color="primary"
  size="lg"
  animation="fadeIn"
/>
```

This renders as:

<AnimatedText
  text="This text animates on page load"
  color="primary"
  size="lg"
  animation="fadeIn"
/>

#### Animation Options

The `AnimatedText` component supports the following options:

-**text**: The text to display and animate
-**color**: `primary`, `secondary`, `success`, `warning`, `danger`, or `gradient`
-**size**: `sm`, `md`, `lg`, `xl`, or `2xl`
-**animation**: `fadeIn`, `fadeInUp`, `fadeInDown`, `fadeInLeft`, `fadeInRight`, `typewriter`, `wave`, `bounce`, `pulse`, or `highlight`
-**delay**: Delay in milliseconds before the animation starts
-**duration**: Duration of the animation in milliseconds
-**easing**: Easing function for the animation (e.g., `easeInOutQuad`, `easeOutElastic`)

### Animated Button

The `AnimatedButton` component creates an animated button with hover effects:

```jsx
import AnimatedButton from '@site/src/components/AnimatedButton';

<AnimatedButton
  text="Click Me"
  color="primary"
  size="md"
  animation="pulse"
  onClick={() => alert('Button clicked!')}
/>
```

This renders as:

<AnimatedButton
  text="Click Me"
  color="primary"
  size="md"
  animation="pulse"
  onClick={() => alert('Button clicked!')}
/>

#### Button Options

The `AnimatedButton` component supports the following options:

-**text**: The button text
-**color**: `primary`, `secondary`, `success`, `warning`, `danger`, or `gradient`
-**size**: `sm`, `md`, `lg`, or `xl`
-**animation**: `pulse`, `bounce`, `shake`, `glow`, or `ripple`
-**onClick**: Function to call when the button is clicked
-**href**: URL to navigate to when the button is clicked (turns the button into a link)
-**target**: Target for the link (e.g., `_blank` to open in a new tab)

## Creating Custom Animations

You can create custom animations using anime.js directly in your MDX files:

```jsx
import React, { useEffect, useRef } from 'react';
import anime from 'animejs/lib/anime.es.js';

function CustomAnimation() {
  const animationRef = useRef(null);
  
  useEffect(() => {
    if (animationRef.current) {
      anime({
        targets: animationRef.current.querySelectorAll('.circle'),
        translateX: function(el) {
          return anime.random(-100, 100);
        },
        translateY: function(el) {
          return anime.random(-100, 100);
        },
        scale: function(el) {
          return anime.random(0.5, 2);
        },
        duration: 2000,
        delay: function(el, i) { return i * 100; },
        direction: 'alternate',
        loop: true,
        easing: 'easeInOutQuad'
      });
    }
  }, []);
  
  return (
    <div ref={animationRef} style={{ height: '200px', position: 'relative' }}>
      <div className="circle" style={{ width: '50px', height: '50px', borderRadius: '50%', backgroundColor: '#3b82f6', position: 'absolute', top: '50%', left: '20%', transform: 'translate(-50%, -50%)' }}></div>
      <div className="circle" style={{ width: '30px', height: '30px', borderRadius: '50%', backgroundColor: '#60a5fa', position: 'absolute', top: '30%', left: '50%', transform: 'translate(-50%, -50%)' }}></div>
      <div className="circle" style={{ width: '40px', height: '40px', borderRadius: '50%', backgroundColor: '#93c5fd', position: 'absolute', top: '70%', left: '70%', transform: 'translate(-50%, -50%)' }}></div>
    </div>
  );
}

export default CustomAnimation;
```

Then use your custom component in your MDX file:

```jsx
import CustomAnimation from './CustomAnimation';

<CustomAnimation />
```

## Animation Best Practices

When adding animations to your documentation, follow these best practices:

1.**Use animations purposefully**: Animations should enhance understanding, not distract from content
2.**Keep it subtle**: Subtle animations are more effective than flashy ones
3.**Consider performance**: Too many animations can slow down page loading and rendering
4.**Respect user preferences**: Always disable animations for users who prefer reduced motion
5.**Ensure accessibility**: Animations should not interfere with screen readers or keyboard navigation

## Respecting User Preferences

Our animation system automatically respects the user's `prefers-reduced-motion` setting. If a user has indicated they prefer reduced motion in their operating system settings, animations will be disabled or simplified.

You can check for this preference in your custom animations:

```jsx
function prefersReducedMotion() {
  return window.matchMedia('(prefers-reduced-motion: reduce)').matches;
}

// In your animation code
if (!prefersReducedMotion()) {
  // Apply animation
} else {
  // Skip animation or use a simplified version
}
```

## Animation Utilities

Our documentation system includes several animation utility functions in the `src/utils/animeUtils.js` file:

```js
// Import animation utilities
import { fadeIn, fadeOut, slideIn, slideOut, pulse } from '@site/src/utils/animeUtils';

// Use in your component
useEffect(() => {
  const element = document.querySelector('.my-element');
  fadeIn(element, { duration: 1000, delay: 300 });
}, []);
```

Available utility functions include:

-**fadeIn/fadeOut**: Fade elements in or out
-**slideIn/slideOut**: Slide elements in or out from different directions
-**pulse/bounce/shake**: Apply attention-grabbing effects
-**typewriter**: Create a typewriter text effect
-**staggered**: Apply staggered animations to multiple elements
-**path**: Animate SVG paths
-**timeline**: Create complex animation sequences

## Advanced Animation Techniques

### Timeline Animations

For more complex animations, you can use anime.js timelines:

```jsx
useEffect(() => {
  const timeline = anime.timeline({
    easing: 'easeOutExpo',
    duration: 750
  });
  
  timeline
    .add({
      targets: '.step-1',
      translateX: 250,
    })
    .add({
      targets: '.step-2',
      translateX: 250,
    })
    .add({
      targets: '.step-3',
      translateX: 250,
    });
}, []);
```

### Scroll-Triggered Animations

You can trigger animations when elements scroll into view:

```jsx
import React, { useEffect, useRef } from 'react';
import anime from 'animejs/lib/anime.es.js';

function ScrollAnimation() {
  const elementRef = useRef(null);
  
  useEffect(() => {
    const observer = new IntersectionObserver(
      (entries) => {
        entries.forEach((entry) => {
          if (entry.isIntersecting) {
            anime({
              targets: entry.target,
              opacity: [0, 1],
              translateY: [50, 0],
              duration: 800,
              easing: 'easeOutQuad'
            });
            observer.unobserve(entry.target);
          }
        });
      },
      { threshold: 0.1 }
    );
    
    if (elementRef.current) {
      observer.observe(elementRef.current);
    }
    
    return () => {
      if (elementRef.current) {
        observer.unobserve(elementRef.current);
      }
    };
  }, []);
  
  return (
    <div ref={elementRef} style={{ opacity: 0 }}>
      This content will animate when it scrolls into view!
    </div>
  );
}

export default ScrollAnimation;
```

### Interactive Animations

You can create animations that respond to user interactions:

```jsx
function InteractiveAnimation() {
  const handleMouseMove = (e) => {
    const x = e.clientX;
    const y = e.clientY;
    
    anime({
      targets: '.follow-cursor',
      left: x,
      top: y,
      duration: 500,
      easing: 'easeOutQuad'
    });
  };
  
  return (
    <div 
      style={{ height: '300px', position: 'relative', overflow: 'hidden' }}
      onMouseMove={handleMouseMove}
    >
      <div 
        className="follow-cursor"
        style={{
          width: '50px',
          height: '50px',
          borderRadius: '50%',
          background: 'radial-gradient(circle, rgba(59,130,246,0.8) 0%, rgba(59,130,246,0) 70%)',
          position: 'absolute',
          transform: 'translate(-50%, -50%)',
          pointerEvents: 'none'
        }}
      ></div>
      <div style={{ padding: '20px' }}>
        Move your mouse over this area to see the effect!
      </div>
    </div>
  );
}
```

## Troubleshooting

If your animations aren't working as expected:

1.**Check Console Errors**: Look for JavaScript errors in the browser console
2.**Verify Element Existence**: Make sure the elements you're targeting exist when the animation runs
3.**Check CSS Conflicts**: CSS properties might be overriding your animations
4.**Test in Different Browsers**: Some animations may behave differently across browsers
5.**Simplify**: Start with a simple animation and build up complexity
